# 足球教练-传奇11人

一个可直接运行的双人足球阵容模拟对战原型。房间、金币、卡包、阵容与比赛结果均由服务端维护；两个浏览器访问同一服务即可通过 6 位房间号对战，也可以邀请 AI 单人体验完整流程。

## 启动

需要 Node.js 18 或更高版本，无第三方依赖。

Windows 可以双击 `start-game.cmd`，或在此目录运行：

```powershell
node server.mjs
```

然后访问：<http://127.0.0.1:4173>

同一局域网好友对战时，可运行：

```powershell
$env:HOST='0.0.0.0'
node server.mjs
```

好友使用你的局域网 IP 加端口 `4173` 访问。部署到公网服务器后，房间号对战流程无需修改。

## 已实现

- 创建好友房、搜索 6 位房间号、双人状态同步、AI 试战
- 每位玩家使用独立会话令牌鉴权，避免同房间内伪造对方操作
- 每局免费提供一套低能力基础 11 人，并给予 5,000 金币用于定向补强
- 8 类卡包、无限购买直至金币不足；不再因抽包顺序而凑不齐首发位置
- 现役包每包 5 张，价格 480–540；传奇包每包 3 张，价格 2,400–2,600
- 200 名五大联赛现役球员与 40 名传奇球员，包含主力、精英、巨星三个现役档位
- 开包动效、逐张翻牌、待定区聚合展示
- 球员卡不设等级与合成，重复抽到的卡牌在待定区按数量聚合
- 同一位置卡池优先抽取尚未拥有的球员，卡包用于提升基础阵容而非购买参赛资格
- 6 种阵型、5 种战术、拖拽或点击放置、一键最佳阵容
- 细分球员擅长位置、位置适配评分、球队默契与战术克制（高位压迫克传控、快速反击克高压）
- 首发同名校验、3 分钟服务端倒计时、双方准备提前开赛
- 第 45、70 分钟开放限时教练调整，可换人并切换进攻、平衡或防守倾向
- 双方阵容看板、实时文字播报、关键事件卡片、简易 2D 球场动画与进球特写
- 球员卡使用更大的中文能力属性，并清晰展示细分擅长位置
- 常规时间、加时赛、点球大战与有限步骤必分胜负
- 赛后比分、决胜方式、MVP、全场数据、关键事件与完整球员评分
- 赛后预期进球（xG）、射门分布图、分阶段战术克制复盘与结构化败因分析；点球大战单独统计
- 断线刷新恢复、再来一局、桌面端与移动端响应式界面

## 公网部署

项目已包含 `render.yaml`、`Dockerfile` 和 `.dockerignore`。

Render 部署：将项目推送到 GitHub、GitLab 或 Bitbucket，在 Render 创建 Blueprint 并选择仓库；服务会通过 `npm start` 启动，自动使用平台提供的 `PORT`，并监听 `0.0.0.0`。

Docker 部署：

```bash
docker build -t football-coach-legend-11 .
docker run -d --name legend-11 -p 4173:4173 football-coach-legend-11
```

## 原型说明

房间数据暂存在服务器内存中，服务器重启后会清空；房间 6 小时无操作自动过期。正式上线时建议接入账号鉴权、数据库、Redis 房间状态和 HTTPS/WSS 部署。球员头像为本地生成的风格化形象，正式商业发行如改用真人照片需单独处理肖像与素材授权。
